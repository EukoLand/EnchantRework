# EnchantmentRework

> [Русская версия](README_ru.md) | English

## Description

This plugin reworks the mechanics of **Bane of Arthropods** and **Smite** enchantments, making them strategically important.

**Without enchantment**: damage to vulnerable mobs is reduced by 30%
**With enchantment**: damage to vulnerable mobs is increased by 30%

This encourages using specialized weapons for different mob types.
