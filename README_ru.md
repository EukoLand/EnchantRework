# EnchantmentRework

> Русская версия | [English](README.md)

## Описание

Плагин переработывает механику зачарований **Бич членистоногих** и **Небесная кара**, делая их стратегически важными.

**Без зачарования**: урон по уязвимым мобам снижается на 30%
**С зачарованием**: урон по уязвимым мобам увеличивается на 30%

Это мотивирует использовать специализированное оружие для разных типов мобов.
